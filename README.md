# Word Guessing Game

A word is randomly chosen from [words.txt](./src/words.txt) and you must guess it in 6 tries.

## Building

`cargo build -r`

## Running

`cargo run`

## Dependencies

This project uses [ratatui](https://ratatui.rs/) and [rand](https://docs.rs/rand/latest/rand/).

See [Cargo.toml](./Cargo.toml)
